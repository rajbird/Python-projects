The Random Cat Facts Django Python project generates random cat facts and displays them on a web page. Users can click a button to get a new random cat fact. The project uses an API to get the cat facts and displays them using a Django template.


*****************************************************
Run Code
*****************************************************

1. Install python and django
2. python manage.py runserver
3. Go to http://127.0.0.1:8000/catfacts/
